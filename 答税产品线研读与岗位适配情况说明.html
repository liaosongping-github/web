一、对在售产品线的理解

一句话：答税做的是「帮财务/税务人员少加班、少出错」的工具 + 「帮人搞懂政策」的内容/AI 平台。

财税 = 财务 + 税务。财务管「账怎么记、钱怎么算」；税务管「该交什么税、怎么向国家申报」。答税两边都碰，但核心更偏「税」相关软件。

1.1 五条产品线

立马审 · 智汇算专业版

帮事务所快速做完「税审」

主要客户：税务师事务所、会计师事务所里的审计/税审人员

税审 ≈ 专业机构帮企业检查「税交得对不对」，并出具正式报告（像体检报告）。企业有时需要这份报告去办某些业务或自查风险。
  • 把企业账套数据导进来（对接很多财务软件）
  • 自动抽查凭证、生成工作底稿
  • 自动算常见「纳税调整」
  • 生成报告，并尽量一键去税局/税协备案
  • 还给事务所看项目进度、回款等经营数据
立马报 · 智汇算企业版

帮企业（或代账）批量把税报出去

主要客户：企业财务、代理记账公司、有很多分子公司的集团

报税 / 纳税申报 ≈ 按规定时间，把「这个月/这个季度该交多少税」算清楚，填表提交给电子税务局（国家收税的网上大厅）。
  • 从财务软件采数 → 自动填表、校验
  • 多税种一起管，减少漏报
  • 多主体（很多公司）可批量申报，少反复登录各地税局
  • 用自动化（含 RPA）减轻人工点鼠标
立马答 · TaxMind

财税版「专业 ChatGPT」

主要客户：会计、税务顾问、企业财务、需要查政策的人

大模型 / AI 问答 ≈ 你用大白话问「研发费用能不能加计扣除」，它结合法规库尽量给出合规、可引用的回答。不是替代专家签字，而是先快速答疑。
  • 绑定答税自建法规库（政策、问答、文章很多）
  • 支持联网追新政策、语音提问、历史对话
  • 常作为会员权益里的「不限次 AI 咨询」
立马查 · 云鹏报告

给企业做「税务+经营风险体检报告」

主要客户：银行等金融机构（放贷风控）、财税机构、企业老板

企业画像 / 风险报告 ≈ 把工商信息、纳税情况、发票异常、欠税处罚、上下游交易等拼成一份「这家公司靠不靠谱」的说明，方便决策。
  • 看税负是否异常、有没有欠税/处罚
  • 看发票红冲作废是否可疑
  • 看供应链集中度、财务健康指标
立马记 · 智能记账

让每月做账更快

主要客户：代理记账公司、中小企业财务

记账 / 凭证 ≈ 把每一笔生意（买东西、卖东西、发工资）记成会计能看懂的「流水账条目」。凭证记对了，后面才报得了税、出得了报表。
  • 取票后尽量自动生成凭证
  • 自动结转、出报表
  • 电子账本，少纸质、少丢数据
平台层 · 获客与知识服务

法规库 / 问答 / 云课堂 / 专家咨询

个人会员 + 机构用户；也是软件产品的「流量入口」

  • 查政策、看课、问专家 → 建立信任
  • 再转化成会员，或升级买智汇算等付费工具
  • 官网还提到「智联平台」（审计资料闭环管理），可作延伸了解

立马审作业流（我对主路径的理解）

采数导账→ 智能抽凭→ 生成底稿→ 纳税调整→ 出报告→ 备案归档
底稿 ≈ 审计/税审过程中留下的「检查记录表」。抽凭 ≈ 从海量记账凭证里按规则挑出要重点检查的那些。纳税调整 ≈ 会计利润和税法认可的利润不一致时，按税法规则改过来再算税。

产品经理价值不在背会每个税种条文，而在于把专家规则沉淀成可配置流程、降低外勤与编制成本，并保证与税局/税协链路稳定。

竞品情况（简版）

立马报 · 报税赛道

常见对照

  • 亿企赢、云账房等代账/报税工具
  • 金蝶 / 用友税务云（偏 ERP 一体化）

答税更偏独立财税作业工具，并可与知识会员、AI 组合。

立马审 · 税审赛道

常见对照

  • 事务所税审作业类软件（如中汇云税审等）
  • 区域/传统税审工具

与金蝶、用友主战场不同：对方强在企业财务与税务管理，税审全流程是更垂直的场景。

我理解的卖点组合

  • 立信专业背景与专家资源
  • 税审 / 申报作业闭环

四、差距认知与补齐计划

坦诚差距

目前仍需加强

  • 常见税种申报节奏与电子税务局操作细节
  • 事务所税审一线的底稿习惯与质控节点
  • 政策更新对产品规则配置的影响机制
补齐方式

可立即执行

  • 与业务/专家共建「规则配置化」，避免把专家绑在每个项目上
  • 用 AI 加速法规与案例检索，人工确认合规边界

五、入职切入建议（供讨论）

在售产品优先「保交付、提效率、促续费」，以下为个人建议，最终以团队优先级为准。

  • 主路径建议：优先深耕一条核心作业产品（立马审或立马报），把主流程体验与关键对接稳定性做扎实。
  • 协同路径：推动 TaxMind / 智能能力嵌入作业节点（答疑、抽凭提示、调整建议），明确「AI 建议 vs 专家签发」边界。
  • 增长路径：理清「内容/会员 → 工具试用 → 付费升级」漏斗，用产品手段降低销售解释成本。
